How Flex Wrap Works

The container above uses these CSS properties:

.flex-container {
    display: flex;
    flex-wrap: wrap;
}

When flex-wrap: wrap is applied:

  • Flex items will stay in a single line until there's not enough space
  • When there's not enough space, items will wrap to the next line
  • This creates a responsive layout that adapts to different screen sizes

Try resizing your browser window to see this in action!
