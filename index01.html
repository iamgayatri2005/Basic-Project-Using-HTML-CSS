Understanding Display Properties

Display: inline

🍔
← This is an inline element

Display: block

🍕
← This is a block element

Display: inline-block

🧁
← This is an inline-block element

Key Differences:

  • Inline: Takes only the space it needs, doesn't force new lines, ignores width/height
  • Block: Takes full width available, forces new lines before and after, respects width/height
  • Inline-block: Flows like inline but respects width/height like block elements
